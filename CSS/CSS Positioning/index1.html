CSS Positions

Static Position - Default HTML Flow

top, right, bottom, left does not work with static

Relative Position

Relative to its current position/location

Absolue Postion

Relative to its nearest placed Ancestor or to the top left corner of the webpage

Z index determines which element goes on top of whichon the z-axis

Element with the higher z-index goes on TOP

Every element has a Default z-index of zero

when you position an element as absolute, it tkes the element out of the default html flow and puts it on another layer

Fixed Position

Relative to the top-left corner of the browser

Even if we scroll UP and DOWN the page it is still going to be located in the same location relative to the browser
